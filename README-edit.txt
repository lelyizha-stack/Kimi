Cara edit:
1. Buka file games.js
2. Tambah data game baru dengan format object yang sama
3. Ganti detail dan download dari "#" ke link aslimu
4. Kalau ingin pakai gambar sendiri, bisa nanti ubah sistem cover di CSS atau di JS

Catatan:
- Search dan filter genre/platform sudah aktif
- Platform dibatasi Windows dan Android
- Halaman kategori otomatis menampilkan data sesuai category
